LINEE DI CODICE

Si può avere visualizzata sul desktop del proprio computer o sul proprio sito web la data de Il Calendario della Terra, così come ad esempio appare di seguito:




tramite un apposito javascript qui fornito, od altro script da voi stessi realizzato. Lo script qui fornito viene così richiamato nel codice HTML:

<SCRIPT language="JavaScript" src="tec_it_1.js">
</SCRIPT>

Lo stile della data può essere modificato usando uno style sheet da inserire nel tag HEAD, proprio come questo che segue:

<STYLE type="text/css">

<!--
a.tec_it_1 {font-family: verdana; font-size: 8pt; font-weight: medium; color: white; background: red; text-decoration: none}
.tec_it_1:hover {color: black; background: yellow; text-decoration: underline}
-->

</STYLE>

Per visualizzare la data sul proprio computer occorre impostare la modalità active desktop. Qui potete vedere un risultato possibile. Questo script viene fornito nelle condizioni in cui si trova, senza alcuna garanzia di buon funzionamento.
